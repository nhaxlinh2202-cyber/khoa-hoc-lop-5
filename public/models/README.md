# 3D Models Directory

Place your `.glb` or `.gltf` model files here.

## Usage

The file will be referenced in the app as `/models/your-model.glb`.

### Current placeholder

The `ActivityThree.tsx` component expects a file named:

```
/models/lactobacillus.glb
```

**To add your model:**
1. Rename your `.glb` file to `lactobacillus.glb`
2. Place it in this folder (`public/models/`)
3. Refresh the browser — the 3D viewer will load automatically

### Supported formats
- `.glb` (Binary glTF) — **recommended**
- `.gltf` (JSON glTF + separate binary/textures)
